# NIFTY Replay Tool (Free)

A simple desktop replay tool for Indian traders to **replay and study NIFTY index price action**  
using historical intraday data.

Built in **Python with a PyQt interface**.

---

## Features

- NIFTY index replay
- Timeframes:
  - 1 minute
  - 3 minute
  - 5 minute
  - 15 minute
  - 3 minute Nifty Futures Data

- Indicators:
  - EMA
  - SMA
  - VWAP
  - RSI
  - MACD
  - BOLLINGER BANDS
  - VOLUME

- Controls:
  - Play / Pause
  - Rewind
- Date selector
- Runs locally on your PC

This tool is meant for **learning and replay only**.

---

## Python Version

- **Python 3.9 or Python 3.10 (recommended)**
- Tested on Python **3.9 / 3.10**
- Python 3.12 is **not recommended** at this time

## One-Click Setup (Windows)

1. Download this repository as ZIP
2. Extract the folder
3. Double-click `setup_run.bat`

### If Python is not installed:
